**Smart Workout IA Coach** es un planificador de rutinas inteligente que utiliza una arquitectura de **Multi-Agentes Inteligentes** para diseñar entrenamientos personalizados basados en biomecánica y motivación deportiva.

---

## 🌟 Características Principales

* **Arquitectura de Agentes (CrewAI):** Un equipo de agentes especializados colabora para crear tu rutina:
    * **Entrenador Senior:** Experto en biomecánica y anatomía.
    * **Coach Motivacional:** Especialista en comunicación y formato de entrenamiento.
* **Memoria Contextual:** El coach recuerda tus mensajes anteriores (pecho, lesiones, objetivos) para adaptar la conversación en tiempo real.
* **Análisis de Sentimiento:** Capacidad para detectar el estado de ánimo del usuario y ajustar el tono de la respuesta.
* **Interfaz Moderna:** Desarrollada en Angular con soporte completo para renderizado de **Markdown** (tablas, negritas y listas en las rutinas).

---

## 🛠️ Tecnologías Utilizadas

### Frontend
* **Angular 18+**: Estructura robusta y componentes Standalone.
* **ngx-markdown**: Para visualizar las rutinas con formato profesional.
* **RxJS**: Gestión de flujos de datos y peticiones asíncronas.

### Backend (IA Service)
* **FastAPI**: API de alto rendimiento para el servicio de inteligencia artificial.
* **CrewAI**: Framework para la orquestación de agentes autónomos.
* **LangChain**: Integración de herramientas de búsqueda (DuckDuckGo).
* **GPT-4o Mini**: El cerebro que procesa las rutinas de forma eficiente.
